# Space Fights!
Most Recent Playable Build: 04.07.2017
Current Classes: Vulture (agile with one shot kill laser but weak), Rhino (tanky bullet-focused ship)

Player 1 Control:
Up: thrust forward
Down: thrust backwards
Left: turn left
Right: turn right
Space: bullet slot
Z: laser slot

Player 2 Controls:
I: thrust forward
J: turn left
K: thrust backwards
L: turn right
Ctrl: bullet slot
Shift: special slot, unused
